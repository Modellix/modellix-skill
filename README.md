# Modellix Agent Skill

An Agent Skill that enables AI agents to access 100+ generative AI models on the Modellix platform.

## What is Modellix?

[Modellix](https://modellix.ai) is a unified AI model API platform that aggregates generative AI models from providers like Alibaba, ByteDance, and MiniMax, supporting:

- **Text-to-Image**: Qwen Image, Wanx, Wan, Seedream, Hailuo, etc.
- **Text-to-Video**: Wan 2.6 T2V, Seedance, Hailuo 02 T2V, etc.
- **Image-to-Video**: Wan 2.6 I2V, Seedance I2V, Hailuo 02 I2V, etc.
- **Image Editing**: Style transfer, watermark removal, image expansion, detail enhancement, etc.
- **Specialized Tasks**: Virtual try-on, image outpainting, image translation, WordArt, etc.
